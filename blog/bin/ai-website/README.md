# Portfolio Website

A minimal, modern personal portfolio built with vanilla HTML, CSS, and JavaScript.

## Features

- Dark theme with accent color
- Smooth scroll-reveal animations
- Responsive layout (mobile, tablet, desktop)
- Sticky nav with scroll-aware styling
- Sections: Hero, About, Projects, Skills, Contact

## Getting Started

Open `index.html` in your browser — no build step required.

```bash
# or use a local server
npx serve .
```

## Customization

| What                | Where                           |
|---------------------|---------------------------------|
| Name & bio          | `index.html` — Hero & About     |
| Projects            | `index.html` — Projects section  |
| Skills              | `index.html` — Skills section    |
| Email & socials     | `index.html` — Contact section   |
| Colors & fonts      | `style.css` — `:root` variables  |
| Profile photo       | Replace the placeholder in About |
